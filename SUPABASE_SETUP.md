# Supabase Configuration for Icon by Mitali

## Environment Variables

Add these to your `.env.local` file:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key
```

## Database Schema

### Orders Table
```sql
CREATE TABLE orders (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  order_number VARCHAR NOT NULL UNIQUE,
  customer_name VARCHAR NOT NULL,
  customer_email VARCHAR NOT NULL,
  customer_phone VARCHAR,
  billing_address JSONB,
  shipping_address JSONB NOT NULL,
  items JSONB NOT NULL,
  total_amount DECIMAL(10, 2) NOT NULL,
  sub_total DECIMAL(10, 2),
  discount DECIMAL(10, 2),
  shipping_charges DECIMAL(10, 2),
  razorpay_order_id VARCHAR,
  razorpay_payment_id VARCHAR,
  shipment_id VARCHAR,
  refund_id VARCHAR,
  status VARCHAR DEFAULT 'pending',
  tracking_details JSONB,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Products Table
```sql
CREATE TABLE products (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  name VARCHAR NOT NULL,
  description TEXT,
  price DECIMAL(10, 2) NOT NULL,
  sku VARCHAR UNIQUE,
  category VARCHAR,
  images JSONB,
  show_in_lookbook BOOLEAN DEFAULT FALSE,
  stock_quantity INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Blog Posts Table
```sql
CREATE TABLE blog_posts (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  title VARCHAR NOT NULL,
  slug VARCHAR NOT NULL UNIQUE,
  content TEXT NOT NULL,
  author VARCHAR,
  featured_image VARCHAR,
  excerpt TEXT,
  published BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### Page Settings Table
```sql
CREATE TABLE page_settings (
  id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
  page_name VARCHAR NOT NULL UNIQUE,
  settings JSONB,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## Supabase Edge Functions Deployment

Deploy the edge functions using Supabase CLI:

```bash
# Install Supabase CLI
npm install -g supabase

# Login to Supabase
supabase login

# Deploy functions
supabase functions deploy razorpay-create-order
supabase functions deploy razorpay-verify-payment
supabase functions deploy razorpay-webhook
supabase functions deploy shiprocket-create-order
supabase functions deploy shiprocket-track
supabase functions deploy get-shipment-details
```

## Environment Variables for Edge Functions

In Supabase Dashboard, add these secrets:

- `RAZORPAY_KEY_ID` - Your Razorpay Key ID
- `RAZORPAY_KEY_SECRET` - Your Razorpay Key Secret
- `RAZORPAY_WEBHOOK_SECRET` - Your Razorpay Webhook Secret
- `SHIPROCKET_TOKEN` - Your Shiprocket API Token

## RLS (Row Level Security) Policies

Set up RLS policies for security:

```sql
-- Enable RLS
ALTER TABLE orders ENABLE ROW LEVEL SECURITY;
ALTER TABLE products ENABLE ROW LEVEL SECURITY;
ALTER TABLE blog_posts ENABLE ROW LEVEL SECURITY;

-- Products are public readable
CREATE POLICY "Products are readable by everyone" ON products
  FOR SELECT USING (true);

-- Published blog posts are readable
CREATE POLICY "Published posts are readable" ON blog_posts
  FOR SELECT USING (published = true);

-- Authenticated users can read their own orders
CREATE POLICY "Users can read their own orders" ON orders
  FOR SELECT 
  USING (auth.uid()::text = customer_email);
```

## Storage Setup

Create storage buckets in Supabase:

1. **public/base44-prod** - For product images and assets (keep existing URLs working)
2. **products** - For product images
3. **blog** - For blog post images
